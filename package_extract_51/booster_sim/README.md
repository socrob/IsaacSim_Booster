# booster_sim

## 启动 issac sim 脚本

先在本地找到 isaac_sim 项目的路径，再通过项目中的 python.sh 来运行启动 isaac sim 项目脚本

以下为示例：
``` 

机器人场景
~/.local/share/ov/pkg/isaac_sim-2023.1.1/python.sh ~/Documents/booster_sim/booster_isaac/booster_standalone.py

robocup 场景
~/.local/share/ov/pkg/isaac_sim-2023.1.1/python.sh ~/Documents/booster_sim/booster_isaac/booster_standalone_robocup.py

```
